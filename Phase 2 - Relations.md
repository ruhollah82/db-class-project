# فاز ۲: روابط

## ویزیت کردن

- موجودیت های درگیر:
  - بیمار
  - پزشک، روان پزشک، دندان پزشک
- کاردینالیتی: چند به چند
- مشارکت: الزامی - الزامی

### صفات رابطه

- کد ویزیت: شناسه، ساده، تک مقداری، ذخیره شده
- کد ملی یا شناسه اتباع بیمار: ساده، تک مقداری، ذخیره شده
- کد نظام پزشکی پزشک: ساده، تک مقداری، ذخیره شده
- تاریخ و ساعت ویزیت: مرکب، تک مقداری، ذخیره شده
- تشخیص بالینی: ساده، چند مقداری، ذخیره شده، هیچ مقدارپذیر
- علائم: ساده، چند مقداری، ذخیره شده، هیچ مقدارپذیر
- فشار خون: ساده، تک مقداری، ذخیره شده، هیچ مقدارپذیر
- ضربان قلب: ساده، تک مقداری، ذخیره شده، هیچ مقدارپذیر
- وزن: ساده، تک مقداری، ذخیره شده، هیچ مقدارپذیر
- دما: ساده، تک مقداری، ذخیره شده، هیچ مقدارپذیر
